\label{ch:executable-crosswalk}

This chapter audits the executable MAgent objective against the probability semantics developed in Chapters~\ref{ch:generative-model}--\ref{ch:information-geometry-gauge}. It reports what the selected click-to-run configuration actually constructs; it does not retrofit a normalized population model onto the implementation. The inspected application snapshot is commit \texttt{3f5f094ab66c4f209fc3eb3c5b35f6f38dc13df0} on \texttt{main}, equal to the fetched \texttt{origin/main} at the time of inspection. The live checkout had documentation-only working-tree changes and no modified executable source. Application code and configuration remained read-only throughout this audit.

\section{The scalar differentiated by the active dynamics}

Let $c$ denote a grid site, $v(c)$ the spherical volume weight, and $\chi_i(c)$ the support field. Reserve $D_{\mathrm{KL}}$ for the floor-free mathematical Gaussian KL. The active executable instead uses $D_\varepsilon$ with $\varepsilon=10^{-4}$. For input covariance tensors, set
\begin{equation}
 S_q=\frac{\Sigma_q+\Sigma_q^{\mathsf T}}{2},
 \qquad
 S_{p,\varepsilon}=\frac{\Sigma_p+\Sigma_p^{\mathsf T}}{2}+\varepsilon I_K,
\end{equation}
and define
\begin{equation}
 \mathcal E_\varepsilon(q\Vert p)
 =\frac12\left[
 \log\frac{\det S_{p,\varepsilon}}{\det S_q}
 +\operatorname{tr}(S_{p,\varepsilon}^{-1}S_q)
 +(\mu_p-\mu_q)^{\mathsf T}S_{p,\varepsilon}^{-1}(\mu_p-\mu_q)-K
 \right].
\end{equation}
After its finite-value and SPD gates, the source applies the bit-identical exact-zero shortcut and final nonnegative clamp. Let $\mathsf Z(q,p)$ denote the source predicate that the input shapes match and that both input pairs pass \texttt{torch.equal}. Then
\begin{equation}
 D_\varepsilon(q\Vert p)=
 \begin{cases}
 0,
 & \mathsf Z(q,p),\\
 \max\{0,\mathcal E_\varepsilon(q\Vert p)\},
 & \text{otherwise}.
 \end{cases}
 \label{eq:executable-divergence}
\end{equation}
The gate and reference-only lift are executable at \path{statistical_manifold.py:268--334}. The exact-zero branch, closed form, and clamp are at \path{statistical_manifold.py:337--415} in the same package. Thus $D_\varepsilon$ is not $D_{\mathrm{KL}}$: the latter is the same Gaussian divergence with no $\varepsilon I_K$ lift and denotes the floor-free object used in the exact theory. Write $w_i(c)=\chi_i(c)v(c)$. For the belief self-coupling, define
\begin{align}
 d_i^{qp}(c) &= D_\varepsilon\bigl(q_i(c)\Vert p_i(c)\bigr),
 &a_i(c) &= \frac{1}{1+d_i^{qp}(c)},\\
 T_1 &= \sum_{i,c}w_i(c)a_i(c)d_i^{qp}(c),
 &R_a &= \sum_{i,c}w_i(c)\left[a_i(c)-\log a_i(c)\right].
\end{align}
Here $b_{0i}=c_{0i}=1$ for every belief fiber. The model self-coupling $T_2+R_{\widetilde a}$ has the same form with
\begin{equation}
 d_i^{sr}(c)=D_\varepsilon\bigl(s_i(c)\Vert r_i(c)\bigr),
 \qquad
 \widetilde a_i(c)=\frac{0.5}{1+d_i^{sr}(c)},
\end{equation}
and
\begin{equation}
 R_{\widetilde a}=\sum_{i,c}w_i(c)\left[
 \widetilde a_i(c)-0.5\log\widetilde a_i(c)\right],
\end{equation}
because $\widetilde b_{0i}=1$ and $\widetilde c_{0i}=0.5$. The implementation anchors are \path{gauge_agent/full_vfe.py:91--172} and \path{gauge_agent/full_vfe.py:617--723}. The precision hyperparameters are detached under the selected policy, but both adaptive coefficients remain differentiable functions of their local divergences.

The belief pair energy and its normalized row prior are
\begin{align}
 E_{ij}^{q}(c)
 &=D_\varepsilon\left(q_i(c)\middle\Vert
    \Omega_{ij}(c)_{\ast}q_j(c)\right),\\
 \widetilde\pi_{ij}^{q}(c)
 &=\frac{\chi_i(c)\chi_j(c)\pi_{ij}^{q}(c)}
 {\sum_{k\ne i}\chi_i(c)\chi_k(c)\pi_{ik}^{q}(c)}.
\end{align}
On a nonempty row the active attention and row functional are
\begin{align}
 \beta_{ij}(c)
 &=\frac{\widetilde\pi_{ij}^{q}(c)
 \exp[-E_{ij}^{q}(c)/\tau_q^{\mathrm{eff}}]}
 {\sum_{k\ne i}\widetilde\pi_{ik}^{q}(c)
 \exp[-E_{ik}^{q}(c)/\tau_q^{\mathrm{eff}}]},\\
 T_3
 &=\sum_{i,j,c}v(c)\left[
 \beta_{ij}(c)E_{ij}^{q}(c)
 +\tau_q^{\mathrm{eff}}\beta_{ij}(c)
 \log\frac{\beta_{ij}(c)}{\widetilde\pi_{ij}^{q}(c)}
 \right],
\end{align}
where $j=i$ is masked and $\tau_q^{\mathrm{eff}}=\tau_q\sqrt K=\sqrt7$. The model term $T_4$ replaces $(q,\beta,\tau_q)$ by $(s,\gamma,\tau_s)$, uses $D_\varepsilon$, and transports through the model frame. The direct support-folding body is \path{gauge_agent/softmax_utils.py:130--175}; the effective-temperature body is \path{gauge_agent/softmax_utils.py:99--127}; the masked row-softmax body is \path{gauge_agent/softmax_utils.py:247--355}; and the elementwise variational objective body is \path{gauge_agent/softmax_utils.py:178--244}. The active $T_3$ and $T_4$ consumers call these bodies at \path{gauge_agent/full_vfe.py:729--811} and \path{gauge_agent/full_vfe.py:817--895}, rather than deriving their semantics from wrapper comments. This is an entropy-retaining, row-normalized variational attention objective. It is not the state-level entropy of a correlated recognition law $Q_\phi(Y\mid o)$ and is not the configuration entropy of Chapter~\ref{ch:configuration-elbo}.

The selected observation source samples one fixed target mean $\mu_i^{\star}\sim\mathcal N(0,I)$ per agent with seed six, broadcasts that mean over the grid, and uses observation precision $\Lambda_o=I$. Since \texttt{resample=False}, each call returns the target itself rather than a new noisy draw. The executable observation energy is
\begin{equation}
 T_5=\frac12\sum_{i,c}w_i(c)\left[
 (o_i(c)-\mu_i^q(c))^{\mathsf T}\Lambda_o(o_i(c)-\mu_i^q(c))
 +\operatorname{tr}\bigl(\Lambda_o\Sigma_i^q(c)\bigr)
 \right].
\end{equation}
The runner constructs the source at \path{run_experiment.py:2627--2655}. Within \texttt{gauge\_agent}, the source constructor is \path{observations.py:139--185} and the fixed-target return is \path{observations.py:296--320}. The energy anchors are \path{full_vfe.py:901--934} and \path{statistical_manifold.py:418--485} in the same package. The tower supplies these observations only to scale zero, so $T_5=0$ at every higher scale (\path{ouroboros.py:3116--3131}). At the base scale this is the parameter-dependent negative expected log likelihood for a fixed normalized Gaussian observation model, with the parameter-independent normalization constant omitted. It does not by itself normalize the remaining population objective.

For each available child--ancestor pair $(i,A)$ at depth $d$, the tower adds
\begin{equation}
 T_6=\sum_{d=1}^{d_{\max}}\rho^d
 \sum_{(i,A),c}w_i(c)
 D_\varepsilon\left(s_i(c)\middle\Vert
 \widetilde\Omega_{iA}(c)_{\ast}s_A(c)\right),
 \qquad \rho=0.5,
\end{equation}
with detached ancestor moments and frames. The active maximum depth is two, the smaller of the VFE depth five and the tower depth two. The term is zero when no higher scale supplies an ancestor and becomes live only after scale formation. Its anchors are \path{gauge_agent/full_vfe.py:940--1126} and \path{gauge_agent/ouroboros.py:623--686}.

The scalar returned as \texttt{total} at each scale is therefore
\begin{equation}
 \boxed{
 F_{\mathrm{live}}
 =(T_1+R_a)+0.5(T_2+R_{\widetilde a})+T_3+0.5T_4+T_5
 +0.5T_6+0.01R_{\mathrm{smooth}}+0.1R_{\mathrm{YM}}.}
 \label{eq:f-live}
\end{equation}
The coefficient assembly is direct source behavior at \texttt{gauge\_agent/full\_vfe.py:2022--2129}. Scale dependence enters through the arguments: $T_5$ is absent above the base, while $T_6$ is absent at the apex and before ancestor formation. $R_{\mathrm{smooth}}$ penalizes belief-frame variation over the grid. At the base scale, $R_{\mathrm{YM}}$ uses the bounded lattice action because the lattice gauge is forwarded. At meta scales without a lattice gauge, the same slot falls back to a continuum penalty formed from vertex frames. The latter is a frame-curvature discretization, not an independent normalized likelihood and not the curvature action of independently parameterized Regime-II links. The source branches are \texttt{gauge\_agent/full\_vfe.py:1164--1383} and \texttt{gauge\_agent/ouroboros.py:499--540}.

Equation~\eqref{eq:f-live} is an engineered, differentiable consensus scalar. The code does not construct a fixed normalized joint $P_\theta(o,Y)$ over all agent states and a correlated normalized $Q_\phi(Y\mid o)$ whose KL identity equals this scalar. The pair energies depend on other live variational states, the adaptive coefficients depend on current divergences, and the closure described below performs a state write outside this scalar. Calling $F_{\mathrm{live}}$ the exact population ELBO of Chapters~\ref{ch:generative-model} and~\ref{ch:structured-recognition-elbo} would therefore be false.

\section{Transport, gauge fixing, and active degeneracies}

Because the manifold-transport flag is false, the T3 and T4 pair transports are vertex-frame coboundaries,
\begin{equation}
 \Omega_{ij}=\Omega_i\Omega_j^{-1},
 \qquad
 \widetilde\Omega_{ij}=\widetilde\Omega_i\widetilde\Omega_j^{-1}.
\end{equation}
Means transform by left multiplication and covariances by the exact congruence
\begin{equation}
 \mu\mapsto\Omega\mu,
 \qquad
 \Sigma\mapsto\Omega\Sigma\Omega^{\mathsf T},
\end{equation}
at \texttt{gauge\_agent/lie\_groups.py:64--94}. This is the executable Regime-I path for attention. The independently stored lattice links remain live for the base-scale $R_{\mathrm{YM}}$ term, so the scalar simultaneously contains a Regime-I attention transport and a Regime-II regularizer. The base links receive updates at rate $0.01$, but they do not transport matter on this path and both neighbor-matter coefficients are zero. Their active role is therefore a gauge self-action channel disconnected from T3, T4, and the disabled neighbor terms.

Smooth initialization occurs before model-fiber tying. The tie operation aliases $\mu_s$, the Cholesky factor of $\Sigma_s$, $\mu_r$, the Cholesky factor of $\Sigma_r$, the model frame, and model precision parameters across all agents at \texttt{run\_experiment.py:1915--1943} and \texttt{gauge\_agent/agents.py:911--980}. Consequently $T_2$ repeats one shared model--hyperprior discrepancy across agents, and all tied model gradients accumulate onto one parameter object.

The same tie degenerates $T_4$, but it does not establish $T_4=0$ under the active literals. The exact-zero shortcut requires the shared model frame to equal the identity. Here \texttt{smooth\_gauge=True} constructs a seeded smooth $SO(K)$ frame before tying. When that generated frame is nonidentity, the selected path takes the representative-pair branch: it computes one shared-state transport residual and broadcasts that value to every off-diagonal pair. Mathematically $\widetilde\Omega\widetilde\Omega^{-1}=I$, but finite-precision solve and congruence operations need not reproduce the input moments bit for bit. Static inspection did not instantiate the frame, so the defensible source-level statement is that $T_4$ is structurally collapsed either to exact zero or to one repeated numerical residual, not that the active source forces a substantive model-disagreement channel. The relevant branches are \texttt{gauge\_agent/full\_vfe.py:817--895}, \texttt{gauge\_agent/agents.py:1015--1123}, and \texttt{gauge\_agent/smooth\_init.py:204--225}.

\section{Simultaneous descent and post-step closure}

At each scale, \texttt{NaturalGradientDynamics.step} evaluates $F_{\mathrm{live}}$, calls \texttt{backward} once, and updates every connected sector in the same step at \texttt{gauge\_agent/dynamics.py:1260--1942}. Belief and prior means use covariance-preconditioned mean updates. Since \texttt{sigma\_natural\_gradient=False}, covariance factors use Euclidean Cholesky-parameter gradients subject to an SPD trust-region check rather than the exact SPD covariance natural-gradient branch. For a belief frame, model frame, or base lattice link $U$, the active Frobenius group update has the form
\begin{equation}
 U^{+}=U\exp\left[-\eta U^{\mathsf T}\nabla_UF_{\mathrm{live}}\right].
\end{equation}
The tied model parameters are updated once after their gradients have been summed through aliasing. The nominal post-$\epsilon$ shared-model rates, before any trust-region rescaling, are $5\times10^{-4}$ for $\mu_s$, $7.5\times10^{-5}$ for its Cholesky factor, and $10^{-4}$ for its frame.

This schedule is not an exact E-step followed by an M-step. The $q$, $p$, $s$, belief-frame, model-frame, and base lattice-twist sectors are exposed to the same backward and update phase; connected variables whose gradients are present are stepped. The $s$ and model-frame steps are multiplied by $\epsilon=0.01$, which supplies a rate separation but not coordinate minimization. The frozen $r$ sector is not updated, precision hyperparameters and temperatures are not learned, and detached ancestors receive no child-scale gradient. No line search or acceptance test checks whether a proposed update lowers equation~\eqref{eq:f-live}.

The Ouroboros tower evolves all existing scales sequentially, checks for scale formation every five steps including step zero, and then invokes self-referential closure. The ordering is anchored at \path{gauge_agent/ouroboros.py:3071--3160}. With the selected frozen-hyperprior policy and at least two scales, closure computes a coherence-weighted aggregate of beliefs and directly overwrites the apex agents' prior means and prior Cholesky factors at \path{gauge_agent/ouroboros.py:2090--2220}. The write occurs after the gradient update and after possible scale formation, on every subsequent tower step rather than only at the five-step formation cadence. A newly formed apex may therefore be closed before it has taken its own dynamics step. The write is not included in the scalar just differentiated, and it cannot affect that same scalar evaluation; it changes later evaluations. Before a higher scale exists, closure is a no-op. Because top-down prior propagation is disabled, no separate downward overwrite runs.

The ordering matters for theory. A closure overwrite can be represented exactly only after it is promoted to a declared probabilistic factor, an accepted coordinate update, or an explicitly analyzed projection. In the selected code it is none of these. Neither simultaneous finite-step descent nor the post-step overwrite carries the monotonicity guarantee proved for exact E updates and accepted generalized-EM M updates in Chapter~\ref{ch:information-geometry-gauge}.

\section{Executable conclusion}

The current runtime preserves several mathematically clean local constructions: normalized Gaussian factors, exact full-covariance congruence transport, normalized variational attention rows, spherical volume weighting, and multiplicative group updates. Those local facts do not compose into the new exact population ELBO. The missing objects are a fixed normalized population joint, a correlated recognition distribution carrying cross-agent dependence, a once-counted configuration law, coordinate or accepted update semantics, and a probabilistic account of apex closure. The proper relationship is therefore a source-traceable comparison: equation~\eqref{eq:f-live} is the scalar the selected application differentiates, while Chapters~\ref{ch:generative-model}--\ref{ch:information-geometry-gauge} define a stricter probabilistic construction that a future implementation could target. No application tests were run because this manuscript task did not change application code.
