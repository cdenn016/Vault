# Refactoring: Improving the Design of Existing Code (2nd ed.)

> [!info] Citation
> Martin Fowler (2018). *Refactoring: Improving the Design of Existing Code (2nd ed.)*. Addison-Wesley, 2nd ed., 2018. ISBN 978-0-13-475759-9.

## TL;DR

The canonical catalog of refactorings — small behavior-preserving code transformations — and the discipline of applying them under test coverage to improve design incrementally.

## Relevance to this research

Software-engineering practice canon informing code-quality standards for the vfe3 implementation (not VFE research theory).
