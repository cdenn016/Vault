# The Pragmatic Programmer (2nd ed.)

> [!info] Citation
> David Thomas, Andrew Hunt (2019). *The Pragmatic Programmer (2nd ed.)*. Addison-Wesley, 20th Anniversary (2nd) ed., 2019. ISBN 978-0-13-595705-9.

## TL;DR

A collection of pragmatic software-craft maxims — DRY, orthogonality, tracer bullets, reversibility — aimed at building adaptable, maintainable systems.
