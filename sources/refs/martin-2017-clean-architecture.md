# Clean Architecture

> [!info] Citation
> Robert C. Martin (2017). *Clean Architecture*. Prentice Hall (Pearson), 2017. ISBN 978-0-13-449416-6.

## TL;DR

Argues software architecture should keep high-level policy independent of low-level details and frameworks, via the SOLID principles and dependency-inversion 'clean architecture' concentric-layer model.

## Relevance to this research

Software-engineering practice canon informing code-quality standards for the vfe3 implementation (not VFE research theory).
