# PEP 484 - Type Hints

> [!info] Citation
> Guido van Rossum, Jukka Lehtosalo, Lukasz Langa (2014). *PEP 484 - Type Hints*. Python Enhancement Proposal 484. https://peps.python.org/pep-0484/

## TL;DR

Introduces a standard syntax and semantics for optional type hints (function annotations) in Python, enabling static type checking without affecting runtime behavior.

## Relevance to this research

Python/PyTorch standards doc informing implementation conventions of the vfe3 codebase.
