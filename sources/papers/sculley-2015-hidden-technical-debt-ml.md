# Hidden Technical Debt in Machine Learning Systems

> [!info] Citation
> D. Sculley, Gary Holt, Daniel Golovin, Eugene Davydov, Todd Phillips, Dietmar Ebner, Vinay Chaudhary, Michael Young, Jean-Francois Crespo, Dan Dennison (2015). *Hidden Technical Debt in Machine Learning Systems*. NeurIPS 2015. https://papers.nips.cc/paper/2015/hash/86df7dcfd896fcaf2674f757a2463eba-Abstract.html

## TL;DR

Argues that ML systems incur large, hard-to-detect 'technical debt' beyond code — entanglement (CACE), hidden feedback loops, undeclared data dependencies, configuration debt, and pipeline jungles — and prescribes practices to contain it.

## Relevance to this research

ML-systems technical-debt taxonomy; engineering-practice context for the codebase.
