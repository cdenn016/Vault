# Reason, Truth and History

> [!info] Citation
> Putnam, Hilary (1981). *Reason, Truth and History*. Cambridge: Cambridge University Press. DOI: [10.1017/CBO9780511625398](https://doi.org/10.1017/CBO9780511625398).

## TL;DR

The book in which Putnam articulates *internal realism*: truth is not correspondence between language and a mind-independent, ready-made world but idealized rational acceptability *within* a conceptual scheme. There is no "God's-eye view," no description of the world from no conceptual standpoint at all; objects and reference are always relative to a framework. The opening "brain in a vat" argument shows, via semantic externalism, that radical skeptical scenarios are self-refuting, motivating the move away from metaphysical realism.

## Problem & setting

Putnam targets *metaphysical realism* — the view that the world consists of a fixed totality of mind-independent objects, that there is one true and complete description of it, and that truth is correspondence to that world. He argues this picture is incoherent (the model-theoretic argument: reference is radically underdetermined) and proposes internal realism as the tenable alternative.

## Method

Philosophical argument: the brain-in-a-vat self-refutation via causal/externalist semantics; the model-theoretic argument that mind-independent reference cannot be uniquely fixed; and a positive account of truth as idealized warranted assertibility relative to a conceptual scheme, with the rejection of any standpoint-free "view from nowhere."

## Key results

- **Internal realism.** Truth = idealized rational acceptability within a scheme; objecthood and reference are scheme-relative.
- **No God's-eye view.** There is no cognitively accessible standpoint outside all conceptual frameworks from which to compare description and world.
- **Brain-in-a-vat self-refutation.** Externalist semantics make the global skeptical hypothesis incoherent.

## Relevance to this research

Putnam's "no God's-eye view" and scheme-relativity of truth are the analytic-philosophy statement of the manuscript's consensus-objectivity / no-view-from-nowhere thesis. The program denies any privileged frame-independent state for an agent; objectivity is what survives across agent-local frames under [[Gauge transformation]], i.e. an internal, intersubjective achievement rather than correspondence to a ready-made world — exactly Putnam's relocation of truth to idealized acceptability within a scheme. This sits beside [[nagel-1986-view-from-nowhere|Nagel 1986]] (the epistemic-limits version of the same claim) and [[vanfraassen-1980-scientific-image|van Fraassen 1980]] (the empiricist stance) in grounding [[Participatory realism (it from bit)]]. It also connects to [[Structural realism]]: if reference is scheme-relative, what travels across schemes is structure, not object-identity — the manuscript's invariant relational content.
