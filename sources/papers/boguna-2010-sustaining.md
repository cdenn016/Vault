# Sustaining the Internet with Hyperbolic Mapping

> [!info] Citation
> Boguñá, Marián, Papadopoulos, Fragkiskos, Krioukov, Dmitri (2010). "Sustaining the Internet with Hyperbolic Mapping." Nature Communications 1, 62 (2010). https://arxiv.org/abs/1009.0267

## TL;DR
Maps the autonomous-system-level Internet into an underlying hyperbolic metric space, showing that greedy routing using only hyperbolic coordinates achieves near-perfect efficiency and resilience. Demonstrates that latent hyperbolic geometry naturally explains network navigability and provides a scalable routing scheme without global topology knowledge.

## Relevance to this research
A foundational hidden-metric-space / hyperbolic-network-geometry paper, cited by García-Pérez 2018 on geometric renormalization; supports the latent-geometry view of network structure relevant to the social-physics project's treatment of network organization.
