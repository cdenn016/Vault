# Belief Updating

Belief updating is the process by which an agent revises its posterior over hidden states given new evidence. In the Bayesian/variational view it is a precision-weighted move from prior to posterior, with the learning rate set by the relative precision of prior versus likelihood; delta-rule and Kalman-style updates are special cases. The program treats deviations from normative Bayesian updating — perseverance, inertia, and momentum in beliefs — as the empirical signature motivating belief-inertia and mass-as-Fisher-information.
