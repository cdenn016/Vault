# Default Mode Network

The default mode network (DMN) is a set of interacting brain regions (medial prefrontal cortex, posterior cingulate/precuneus, angular gyrus) most active during rest, self-referential thought, and mind-wandering. Its disintegration under psychedelics correlates with ego dissolution and an 'entropic brain' state of increased neural signal diversity, making it a focal structure in theories linking self-modeling, predictive processing, and consciousness. In the program it grounds the neural side of self-model / hierarchical-agent and entropic-brain discussions.
