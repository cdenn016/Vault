# Inbox

Landing folder for **new / unsorted notes**. Obsidian is configured (`.obsidian/app.json`:
`newFileLocation: folder`, `newFileFolderPath: inbox`) to create new notes here rather than at the
vault root, so accidental stubs from clicking an unresolved wikilink land in one obvious place
instead of polluting the root.

Triage anything that accumulates here into the proper layer — `sources/` (immutable raw notes),
`wiki/` (synthesis), or delete if it was an accidental empty stub. See [[LLM-Wiki Schema]].
